# my_homelab_project

Коллекция проектов домашней лаборатории: инфраструктура, автоматизация, самохостинг.

## Проекты

| Папка | Описание |
|-------|----------|
| [terraform-k3s](./terraform-k3s/) | Автоматическое развёртывание K3s-кластера на Proxmox через Terraform |

## Стек

- **Гипервизор:** Proxmox
- **Оркестрация:** K3s (Kubernetes)
- **IaC:** Terraform
- **CI/CD:** GitLab

## Как добавляются проекты

Проекты публикуются автоматически из локального GitLab через pipeline.
Каждый проект — отдельная папка в этом репо.
